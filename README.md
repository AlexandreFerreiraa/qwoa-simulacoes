# Quantum Walk-based Optimization Algorithm (QWOA)
Simulação via QISKIT do circuito QWOA usando portas universais para otimização de alocação simples de ativos financeiros.  
Este é um trabalho realizado para fins avaliativos da disciplina de computação quântica em 2025/2 - UFRGS.
